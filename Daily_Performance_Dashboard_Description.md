# SZTrades Daily Performance Dashboard - Trading Journal Tracker

## 📊 Overview
**SZTrades Daily Performance Dashboard** is a comprehensive real-time trading performance tracker designed for active day traders and scalpers. This overlay indicator provides instant visual feedback on your daily trading metrics without cluttering your chart space.

## 🎯 Key Features

### **📈 Real-Time Performance Tracking**
- **Daily P&L**: Live profit/loss tracking with color-coded display
- **Win Rate**: Percentage of winning trades updated in real-time  
- **Risk/Reward Ratio**: Average R/R across all daily trades
- **Max Drawdown**: Peak-to-trough loss measurement in dollars
- **Trade Count**: Total number of executed trades

### **📋 Detailed Trade Analysis**
- **Average Win/Loss**: Separate calculations for winning and losing trades
- **Best/Worst Trade**: Extreme performance tracking for risk assessment
- **Average Stop Loss Size**: Risk management metric
- **Average Holding Time**: Trade duration analysis in minutes

### **⚙️ Manual Input System**
Perfect for live trading platforms like Tradovate, NinjaTrader, or any broker:
- **Trade Result**: Win/Loss selection
- **P&L Entry**: Dollar amount input
- **R/R Ratio**: Risk/reward tracking
- **Stop Loss Size**: Risk measurement
- **Holding Time**: Duration in minutes

### **🕐 Smart Daily Reset**
- **6:00 PM ET Reset**: Captures Asia, Pre-market, RTH, and After-hours sessions
- **Full Session Coverage**: From Asia open to US close
- **Weekend Handling**: Automatic new day detection

## 🎮 How to Use

### **Setup**
1. Add indicator to your trading chart
2. Enable "Manual Trade Input Mode" in settings
3. Position tables using "Table Position" dropdown

### **Live Trading**
1. After each trade, input the result:
   - Select "Win" or "Loss"
   - Enter P&L amount in dollars
   - Add R/R ratio and stop loss size
   - Input holding time in minutes
2. Dashboard updates automatically
3. Monitor real-time performance metrics

### **End of Day**
- Take screenshot for journal records
- Copy data to Excel/Google Sheets
- Dashboard resets automatically at 6:00 PM ET
