# SZTrades Multi-Timeframe Alignment + Avg. ATR - Professional Confluence Tool

## 📊 Overview
**SZTrades Multi-Timeframe Alignment + Avg. ATR** combines two essential trading elements: precise volatility measurement through Average True Range (ATR) and real-time multi-timeframe directional analysis. This powerful confluence tool helps traders identify high-probability setups by aligning market direction across multiple timeframes while providing accurate target estimation.

## 🎯 Key Features

### **📈 Advanced ATR Calculation**
- **Session-Based ATR**: Calculates average True Range from market open (9:30 AM) onwards
- **Configurable Periods**: Adjustable ATR calculation periods (default: 4 periods)
- **Target Estimation**: Automatic target calculation with 0.5x multiplier for realistic profit targets
- **Real-Time Updates**: Live ATR values updated throughout the trading session

### **🚦 Multi-Timeframe Alignment (Traffic Light System)**
Visual semaphore showing current directional bias across four key timeframes:
- **4H**: Primary trend direction (institutional timeframe)
- **1H**: Intermediate trend confirmation  
- **30M**: Short-term momentum assessment
- **15M**: Precise entry timing analysis

Each timeframe displays **BULL** (green) or **BEAR** (red) based on current candle close vs open.

### **⚡ Professional Display**
- **Clean White Table**: Professional appearance with clear hierarchy
- **Compact Layout**: ATR data on top, MTF alignment below
- **Customizable Position**: 9 table positioning options
- **Toggle Controls**: Enable/disable individual components
